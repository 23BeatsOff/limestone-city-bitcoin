# Limestone City Bitcoin 

A community-led initiative dedicated to Bitcoin education and adoption in Kingston, Ontario, and the surrounding area. 

This project provides a high-signal directory of Canadian infrastructure, educational resources, and a guide to understanding Bitcoin through the lens of local economic realities.
